# Descripción General

Esta aplicación de Punto de Venta (POS) está diseñada para ofrecer una solución de escritorio completa y eficiente para la gestión de ventas, inventario y operaciones comerciales. La aplicación está construida con Flutter y aprovecha Riverpod para la gestión del estado, garantizando una arquitectura robusta, escalable y mantenible.

## Estilo y Diseño

- **Tema:** Moderno y limpio, con una paleta de colores personalizable.
- **Fuentes:** Tipografía clara y legible para una fácil lectura.
- **Diseño:** Intuitivo y fácil de usar, con un diseño responsivo que se adapta a diferentes tamaños de pantalla.

## Características

- **Autenticación:** Inicio de sesión seguro para administradores y cajeros.
- **Gestión de Productos:**
  - Añadir, editar y eliminar productos.
  - Organizar productos por categorías, marcas y departamentos.
- **Gestión de Inventario:**
  - Múltiples almacenes.
  - Seguimiento de stock.
- **Ventas:**
  - Interfaz de caja registradora.
  - Procesamiento de ventas.
- **Gestión de Entidades:**
  - Proveedores.
  - Clientes (próximamente).
- **Configuración:**
  - Tasas de impuestos.
  - Métodos de pago (próximamente).
